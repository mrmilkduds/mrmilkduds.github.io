Problem

Mob Boss is an application written for use in Discord. Its main purpose is to provide a system that incentivizes community engagement through a text-based game that focuses on displayable stats. There needed to be a wide variety of stats that players could pursue excellency in - Bob might have the rarist mob in the server, but I have a mob with perfect stats! This project could best be defined by the following categories.

  • Game Logic
  • Game Info
  • Game Display
